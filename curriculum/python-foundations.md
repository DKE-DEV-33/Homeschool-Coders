# Python Foundations

## Audience split

### Little Learners

This track should teach programming through motion, color, and repeated patterns.

1. Move the turtle
2. Turn and make corners
3. Repeat to make patterns
4. Change colors
5. Create helper commands
6. Use simple `if` choices

### Explorer

This track should mirror the same ideas but explain the language mechanics more clearly.

1. Statements and order of execution
2. Parameters and visual side effects
3. Loops for geometric repetition
4. Functions as reusable abstractions
5. Conditionals and branching behavior
6. Small projects combining multiple ideas

## First lesson sequence

### Mission 1

- Goal: make one line
- Concepts: command, direction, visible feedback
- Checkpoint: draw a straight path to a target marker

### Mission 2

- Goal: make a square
- Concepts: turns, angle, repetition
- Checkpoint: explain why four turns of ninety degrees closes the shape

### Mission 3

- Goal: make a flower pattern
- Concepts: loops, color, experimentation
- Checkpoint: change the loop count and predict the result

### Mission 4

- Goal: make a reusable shape helper
- Concepts: functions, naming, reuse
- Checkpoint: use the helper more than once to build a scene
